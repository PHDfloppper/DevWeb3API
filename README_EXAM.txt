Procédure d'installation:
simplement cloner le repo et faire npm run dev et tout devrait marcher.
cela dit, l'api est set pour fonctionner avec la bd distante, pas local. Si on veut utiliser la bd local,
il faut modifier le lien dans les variables d'états

Procédure de création de bd:
Dans MongoDbCompass, créer un bd inventaires avec une collection Inventaire avec les données de
inventaires.Inventaires.json dans le dossier Dev

Url de l'api: https://olidevwebapi.netlify.app/api-docs/

ma documentation d'api a pas l'air de fonctionner sur netlify mais en local elle marche.
